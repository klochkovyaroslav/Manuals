# NANO

#### Запустите режим поиска и замены

```bash
Ctrl + \
```

Внизу экрана появится запрос: Search (to replace) :  
Введите слово или текст, который хотите найти.  
Нажмите Enter.  

Введите текст для замены  
Появится новый запрос: Replace with :  
Введите новый текст  
Нажмите Enter  

Как подтверждать замены  
Nano спросит для каждого найденного совпадения:  
Y (Yes) — заменить это вхождение и перейти к следующему  
N (No) — пропустить это вхождение  
A (All) — заменить все вхождения сразу (без дополнительных вопросов)  
Ctrl + C — отменить операцию
