## fsfreeze в Linux: Заморозка файловой системы для консистентных снапшотов


#### Как посмотреть события fsfreeze в Linux
```bash
sudo joutnalctl -b | grep -i freeze
sudo journalctl -b | grep fsfreeze
sudo dmesg | grep -i freeze
```
